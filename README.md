## About the Course
5-Day AI Agents Intensive 是一門由 Google 的 AI 研究員與工程師所設計的免費且結構化的線上課程。它涵蓋了 AI Agents 的理論基礎與動手實作，從簡單的 Agents 逐步演進到完整的多 Agent 系統（Multi-Agent Systems）。本課程特別強調 Vibe Coding 工作流，讓自然語言成為主要的程式設計介面，並引導學習者如何透過整合工具、API 與多 Agent 系統來打造「10x Agents」。

---

## Course Structure

| Day | Topic | Key Concepts |
| :--- | :--- | :--- |
| **Day 1** | **Introduction to Agents & Vibe Coding** | 從 AI 聊天機器人和文本補全升級為自主 Agents。掌握以自然語言作為主要程式設計介面的 Vibe Coding 工作流。 |
| **Day 2** | **Agent Tools & Interoperability** | 藉由整合外部 API、程式碼執行（Code Execution）以及 Agent 之間的通訊（Agent-to-Agent Communication）來探索無限可能。 |
| **Day 3** | **Agent Skills** | 建構具有長期記憶與狀態的個人化 Agents。掌握長上下文（Long Context）與最佳 Token 使用策略，並建構可整合至 Agentic 框架中的 Agents 技能。 |
| **Day 4** | **Vibe Coding Agent Security and Evaluation** | 透過實施嚴格的測試、護欄（Guardrails）與品質評估，開發出可靠的 Agents。防範 Agents 遭受新型威脅向量的攻擊。 |
| **Day 5** | **Spec-Driven Production Grade Development in the Age of Vibe Coding** | 將您的本地 Agents 升級為受治理、可擴展且可觀測的生產級 Fleet。掌握雲端部署、偵錯（Debugging）與觀測性（Observability）。 |

## Tech Stack

- **LLMs**: Google Gemini (Gemini-Pro, Gemini-Flash)
- **Frameworks**: LangChain, LangGraph, Google Gen AI SDK
- **Vector Stores**: ChromaDB / FAISS
- **Environment**: Python, Jupyter Notebooks

## Prerequisites
- **Python 程式設計**：本課程包含動手編寫程式碼的練習，因此強烈建議具備 [Python](https://www.kaggle.com/learn/python) 基礎。
- **基本 AI 概念**：具備基本 AI 概念將有助於學習。如果您是新手，建議先參考前一次的 [5-Day AI Agent Intensive Course with Google](https://www.kaggle.com/learn-guide/5-day-agents)。
- **Google AI Studio**：熟悉 [Google AI Studio](https://aistudio.google.com/) 有助於有效地實驗與部署 AI 模型。請確保您所在的地區支援 Gemini API，詳見[可用區域清單](https://ai.google.dev/gemini-api/docs/available-regions)。
- **Kaggle 帳號**：需擁有已驗證的 Kaggle 帳號。

## Setup Instructions
1. 建立並驗證您的 [Kaggle 帳號](https://www.kaggle.com/)。
2. 進行 Kaggle 手機號碼驗證（以存取程式碼實驗室與參與 Capstone 專案）：
   - 點擊 Kaggle 個人資料頁面右上角的頭像，選擇 "Settings"。
   - 在 "Phone Verification" 下點擊 "Verify Phone"。
   - 按照畫面說明完成驗證（請包含國家代碼，且前面不要有額外的數字或符號）。
3. 設定 [Google AI Studio](https://aistudio.google.com/) 並生成 API 金鑰。
4. 加入 [Kaggle Discord 社群](http://discord.gg/kaggle) 專屬頻道進行討論與交流。
