# TechNews CLI Tool

> 2026-07-11T15:02:00Z

## Background and Problem
Create a CLI tool written in Node.js to fetch latest headlines from https://technews.tw/ and allow the user to open them in their browser from the terminal.

## Attempted Solutions
- Set up a package.json configured with ES modules.
- Choose dependencies: `cheerio` for parsing, `picocolors` for terminal colors, `open` for launching standard browser.
- Use native `fetch` API (Node 24).
- Create bilingual documentation (Traditional Chinese and English) in separate files (`README.md` and `README.en.md`) mapping the scraper logic and interactive menu functionality.

## Results
- Project initialized, package.json created, and dependencies installed successfully.
- `index.js` implemented with Cheerio for HTML parsing, `seenUrls` Set for deduplication, and a recursive `readline` prompt loop for user terminal inputs.
- Created `README.md` in Traditional Chinese and `README.en.md` in English with thorough installation, usage guide, and technical details.
- Checked cross-links and project structures.
