# Day 1 (Introduction to Agents & Vibe Coding)

Welcome to Day 1.

這裏探討了從手動語法編碼（manual syntax coding）到意圖驅動的 "vibe coding" 以及規律的 "agentic engineering" 的轉變。它探索了 AI agents 如何壓縮傳統的軟體開發生命週期（SDLC），並解釋了「工廠模式」（factory model）——在此模式中，開發人員扮演系統協調者的角色，負責設計評估、約束與上下文的安全機制，以安全地引導自主執行。

在 codelabs 中，您將熟悉 Antigravity 2.0、Antigravity IDE 以及 Antigravity CLI，以 vibecode 您的第一個應用程式。您還將使用 Google AI Studio 將您的應用程式部署到 Cloud Run，從而能與您的朋友們分享您的 vibe coded 應用程式。
